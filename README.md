This portfolio is automatically deployed to GitHub Pages using GitHub Actions.

### Automatic Deployment

1. Push to the `main` branch triggers automatic deployment
2. The workflow builds and deploys the site
3. Site will be available at: `https://[username].github.io/portfolio/`

### Manual Deployment Setup

1. **Enable GitHub Pages in your repository:**

   - Under "Build and deployment", select "GitHub Actions" as the source

2. **Push to main branch:**
   ```bash
   git add .
   git commit -m "Deploy portfolio"
   git push origin main
   ```

3. **Monitor deployment:**
   - Check the Actions tab in your repository
   - Wait for the workflow to complete
   - Your site will be live at the GitHub Pages URL
